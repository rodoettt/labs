**Менеджер файлов**

Выполнил: Котоманов Тимофей 
Группа: КИ23-02/3Б

**Описание приложения**

Приложение предназначено для парсинга, хранения и обработки информации о файлах. Каждая запись содержит три поля:
- **Имя файла** (строка)
- **Дата создания** (в формате `ГГГГ.ММ.ДД`)
- **Размер файла** (целое число в байтах)

**Основные возможности**

-  **Ручной ввод** - парсинг одиночных записей о файлах
- **Загрузка файлов** - массовая загрузка данных из текстовых файлов
-  **Сохранение данных** - экспорт всех записей в файл
-  **Командный режим** - выполнение пакетных команд (ADD, REM, SAVE)
-  **Логирование ошибок** - детальный журнал ошибок с возможностью экспорта
-  **Модульные тесты** - встроенное тестирование перед запуском

### Требования

- CMake 3.16 или выше
- Qt 6 с компонентами: Core, Widgets, Test
- Компилятор с поддержкой C++17

При запуске ./file_app автоматически выполняются модульные тесты. Если тесты не пройдены, приложение не запустится (если не указан флаг --skip-tests).
